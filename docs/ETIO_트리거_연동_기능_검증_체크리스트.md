# ETIO 트리거 연동 기능 검증 체크리스트

작성일: 2026-04-01
대상: ETIO-2444 + Sony Camera + Q-camera SDK 연동 프로그램
목적: 바이브 코딩으로 구현한 프로그램의 기능 연동, 예외 처리, 안정성 동작을 사용자가 직접 점검하고 완료 여부를 쉽게 표시할 수 있도록 함

## 사용 방법
- `체크` 칸에 `[ ]`를 `[x]`로 바꾸면 완료 표시가 됩니다.
- `결과/메모` 칸에는 실측 결과, 오류 내용, 특이사항을 기록합니다.
- 필요 시 테스트 일자, 테스트 환경, 담당자를 문서 상단에 추가해서 사용합니다.

## 2. ETIO 입력/출력 및 인터럽트 검증

| 체크 | ID | 검증 항목 | 확인 방법 | 기대 결과 | 결과/메모 |
| --- | --- | --- | --- | --- | --- |
| [ ] | I-01 | IO Input 상태 읽기 | 각 Input 채널에 신호를 순차적으로 인가한다 | 프로그램에 각 채널 상태가 정확히 반영된다 | |
| [ ] | I-02 | IO Output 제어 | 프로그램에서 Output 채널을 ON/OFF 한다 | 실제 Output 상태와 화면 표시가 일치한다 | |
| [ ] | I-03 | Rising Edge 인터럽트 | Interrupt Polarity를 Rising으로 설정 후 신호를 인가한다 | Rising Edge에서만 이벤트가 발생한다 | |
| [ ] | I-04 | Falling Edge 인터럽트 | Interrupt Polarity를 Falling으로 설정 후 신호를 인가한다 | Falling Edge에서만 이벤트가 발생한다 | |
| [ ] | I-05 | All Edge 인터럽트 | 양 방향 Edge 검출 모드로 설정 후 신호를 인가한다 | Rising/Falling 모두 이벤트가 발생한다 | |
| [ ] | I-06 | 인터럽트 비활성화 | Interrupt Disable 후 신호를 인가한다 | 이벤트가 발생하지 않는다 | |

## 3. Encoder 및 Trigger 기능 검증

| 체크 | ID | 검증 항목 | 확인 방법 | 기대 결과 | 결과/메모 |
| --- | --- | --- | --- | --- | --- |

| [ ] | T-02 | Virtual Encoder 동작 | Virtual Encoder 주파수를 설정하고 동작시킨다 | 설정값에 맞는 Trigger가 생성된다 | |
| [ ] | T-03 | IO Bypass 동작 | IO Bypass 모드에서 입력 신호를 인가한다 | 입력 신호가 Trigger 출력으로 전달된다 | |
| [ ] | T-04 | Normal Trigger 모드 | Normal 모드로 설정 후 Encoder 입력을 준다 | 연속 Trigger가 정상 생성된다 | |
| [ ] | T-05 | Range Trigger 모드 | Range 모드와 Count Range를 설정 후 신호를 인가한다 | 지정된 범위 내에서만 Trigger가 생성된다 | |
| [ ] | T-06 | Working Area Type 1 | Type 1 설정 후 센서 입력과 엔코더 입력을 준다 | 센서 유효 구간 동안만 Trigger가 발생한다 | |
| [ ] | T-07 | Working Area Type 2 | Type 2 설정 후 테스트한다 | Edge 이후 지정 범위 동안 Trigger가 발생한다 | |
| [ ] | T-08 | Working Area Type 3 | Type 3 설정 후 Sensor 1, Sensor 2, Reset 조건을 테스트한다 | 지정 구간에서만 Trigger가 발생한다 | |
| [ ] | T-09 | Pulse Ratio 반영 | Pulse Ratio 값을 변경하면서 테스트한다 | 출력 Trigger 간격이 설정값에 맞게 변한다 | |
| [ ] | T-10 | Output Polarity 반영 | Output Polarity를 변경하여 테스트한다 | 출력 극성이 설정대로 변경된다 | |
| [ ] | T-11 | Output Width 반영 | Output Width를 여러 값으로 변경한다 | 출력 폭이 설정값대로 반영된다 | |
