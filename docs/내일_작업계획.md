# 내일 작업 계획 (UI_stitch)

작성일: 2026-04-01
브랜치: UI_stitch

## 목표
- Windows 11 스타일 UI를 안정화하고 기능 연결 검증 완료
- 설치 패키지(설치/제거 포함) 재검증
- 문서/성과물 정리 후 master 반영 준비

## 1순위 (오전)
1. CameraDemo 빌드/실행 확인
2. UI 주요 동작 확인
   - Start Preview / Record / Batch Save
   - Exposure/Gain/Focus 즉시 반영
   - Control Panel 파라미터 반영
3. 로그 파일 생성/복사 동작 확인

## 2순위 (오전~점심)
1. 디자인 디테일 보정
   - 여백/타이포/색상 대비
   - 버튼 상태 색상 통일
   - 패널 스크롤/작은 해상도 대응
2. 오버레이/상태 텍스트 정리

## 3순위 (오후)
1. 설치 패키지 재생성
2. 설치/제거(앱 및 기능 항목 포함) 재검증
3. 문서 업데이트
   - README
   - docs/성과물_목록.md
   - SDK/ATP 문서

## 산출물 체크
- UI_stitch 빌드 성공 로그
- 설치 파일: dist/installer/QCameraSDK_Setup.exe
- 문서 최신화 완료

## 리스크
- 카메라 점유로 인한 0x80070005
- 외부 카메라 앱 동시 실행에 따른 FPS/점유 간섭
